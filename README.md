## üìÇ File Structure Overview

All tasks are maintained as markdown files named by day for clarity and traceability.

* `Day 01` to `Day 100` ‚Äî Individual DevOps tasks with commands, configurations, and outcomes
* `Commands (from 50 Days of DevOps).md` ‚Äî Frequently used Linux and DevOps commands for quick reference
* `Terminology (From 50 Days of DevOps).md` ‚Äî Common DevOps and cloud terminology explained concisely
* `SS/` ‚Äî Screenshots and visual references related to selected tasks
